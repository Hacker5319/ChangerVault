# ChangerVault

Плагин для Paper 1.21+, позволяющий изменять шансы выпадения предметов из хранилищ (Vault) в испытательных палатах.

## Возможности

- Изменение шанса выпадения любых предметов из обычных и зловещих хранилищ
- Добавление новых предметов в лут-таблицу хранилищ
- Включение/выключение плагина через команду
- Анимированный выброс предметов с звуками и частицами
- Гибкая настройка через конфигурационные файлы

## Установка

1. Скачайте `ChangerVault.jar`
2. Поместите файл в папку `plugins` вашего сервера
3. Перезапустите сервер или выполните `/reload confirm`
4. Настройте шансы в `plugins/ChangerVault/tables.yml`
5. Готово!

## Команды

| Команда | Алиасы | Описание | Права |
|---------|--------|----------|-------|
| `/changervault` | `/cv` | Включает/выключает плагин | `changervault.toggle` |

## Права

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `changervault.toggle` | Позволяет переключать режим работы плагина | OP |

## Конфигурация

### config.yml
```yaml
activated: true

messages:
  no-permission: " &f[&cОшибка&f]&c У вас недостаточно прав"
  enabled: " &f[&aУспех&f]&a Режим изменения шансов включен"
  disabled: " &f[&cОшибка&f]&c Режим изменения шансов выключен"
```
### tables.yml
```yaml
mace:
  type: "OMINOUS"
  material: "MACE"
  chance: 15

diamond_ominous:
  type: "OMINOUS"
  material: "DIAMOND"
  chance: 50

emerald_common:
  type: "COMMON"
  material: "EMERALD"
  chance: 30
```
## Параметры tables.yml:

type — тип хранилища:
COMMON — обычное хранилище
OMINOUS — зловещее хранилище
material — материал предмета (например: MACE, DIAMOND, HEAVY_CORE)
chance — шанс выпадения в процентах (0-100)
